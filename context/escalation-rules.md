# Escalation Rules

## Automatic Escalation Triggers
1. Customer mentions "lawyer", "legal", "sue", "attorney"
2. Sentiment score < 0.3 (very negative)
3. More than 2 failed knowledge base searches
4. Customer explicitly asks for human
5. WhatsApp: customer sends "human", "agent", "representative"
6. Pricing negotiation requests
7. Refund requests over \
8. Data breach or security concerns
9. GDPR/compliance questions
10. Enterprise contract modifications

## Escalation Process
1. Acknowledge customer immediately
2. Create high-priority ticket
3. Notify on-call agent via Slack #urgent-support
4. Send customer: "A specialist will contact you within 1 hour"
5. Include full conversation context in escalation
