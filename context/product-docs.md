# TeamFlow Pro - Product Documentation

## Getting Started
### Account Setup
1. Sign up at app.techcorp.io
2. Verify your email
3. Create your first workspace
4. Invite team members

## Core Features

### Projects
- Create projects with custom boards
- Use Kanban, Scrum, or List view
- Set deadlines and milestones

### Tasks
- Assign tasks to team members
- Set priority: Low, Medium, High, Urgent
- Add subtasks, attachments, comments
- Track time with built-in timer

### Messaging
- Team channels and direct messages
- File sharing up to 100MB
- Video call integration (Zoom, Meet)

## Common Issues & Solutions

### Password Reset
1. Go to app.techcorp.io/login
2. Click 'Forgot Password'
3. Enter your email
4. Check inbox for reset link (expires in 24h)

### API Access
- Generate API keys in Settings > Developer
- Rate limit: 1000 requests/hour
- Documentation: docs.techcorp.io/api

### Billing
- Invoices sent on 1st of each month
- Payment methods: Credit card, bank transfer
- Refunds: Contact billing@techcorp.io

### Integrations
- Slack: Settings > Integrations > Slack
- GitHub: Connect via OAuth in Settings
- Jira: Use API key from Atlassian account
